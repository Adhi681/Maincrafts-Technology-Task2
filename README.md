# Maincrafts-Technology-Task2
Multi-page responsive website built with HTML, CSS & JavaScript

## Live Demo
https://github.com/Adhi681/Maincrafts-Technology-Task2/

## About
This is a multi-page website project built as part of the Full Stack Web Development Internship at MainCrafts Technology. It expands on Task 1 by adding multiple connected pages, JavaScript form validation, and a shared navigation system across all pages.

## Pages
- **Home** — Hero section, Features, Stats
- **About** — Company info, Values, Team
- **Contact** — Contact form with JavaScript validation

## Tech Stack
- HTML5
- CSS3
- JavaScript (Form Validation)

## How to Run Locally
- Clone this repository
```
git clone https://github.com/Adhi681/Maincrafts-Technology-Task2.git
```
- Open `index.html` in your browser
